# Image Background Remover - MVP 需求文档

## 1. 项目概述

| 项目 | 内容 |
|------|------|
| **项目名称** | Background Remover MVP |
| **项目类型** | Web 应用 (图片处理工具) |
| **核心功能** | 上传图片，自动移除背景，返回透明 PNG |
| **目标用户** | 电商卖家、设计师、需要透明背景图片的用户 |
| **部署平台** | Cloudflare Workers |
| **图片处理** | 内存处理，无持久化存储 |
| **底层 API** | Remove.bg API |

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽或点击上传图片 (PNG/JPG/WEBP, max 10MB) | P0 |
| 背景移除 | 调用 Remove.bg API 移除背景 | P0 |
| 结果预览 | 展示原图 vs 处理后对比 | P0 |
| 图片下载 | 一键下载透明背景图片 | P0 |

### 2.2 扩展功能 (后续)

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 批量处理 | 一次上传多张图片 | P1 |
| 历史记录 | 最近处理的图片 (内存缓存) | P1 |
| 下载原图+蒙版 | 同时下载原图和蒙版 | P2 |

## 3. 非功能需求

### 3.1 性能

| 指标 | 要求 |
|------|------|
| 响应时间 | < 10s (取决于 Remove.bg API) |
| 并发限制 | 同 IP 每分钟 5 次 |
| 图片大小 | 最大 10MB |
| 支持格式 | PNG, JPG, JPEG, WEBP |

### 3.2 成本

| 项目 | 预估 |
|------|------|
| Cloudflare | 免费额 (Workers + Bandwidth) |
| Remove.bg | 免费 tier: 50张/月 |

### 3.3 安全

- API Key 存储于 Cloudflare Secrets
- 不存储用户图片 (内存处理)
- 限制上传文件类型
- 限制文件大小

## 4. 技术架构

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   前端页面   │ ──▶ │ Cloudflare  │ ──▶ │ Remove.bg  │
│  (HTML/CSS)  │     │   Workers   │     │    API     │
└─────────────┘     └──────────────┘     └─────────────┘
      │                    │                     │
      │              API Key (Secrets)          │
      │                    │                     │
      ▼                    ▼                     ▼
   用户浏览器          内存处理              返回透明图
```
